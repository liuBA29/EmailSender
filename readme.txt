# Рассыльщик писем

Этот проект представляет собой простое приложение для рассылки писем на основе списка адресов электронной почты, предоставленного в файле Excel. Программа позволяет добавлять вложения и отправлять письма с указанной темой и текстом.

## Требования

- Python 3.6 или выше
- Файл Excel с адресами электронной почты

## Установка

1. **Скачайте проект:**
   - Скачайте файлы проекта из репозитория или архива и распакуйте их в нужную директорию на вашем компьютере.

2. **Установка зависимостей:**
   - Откройте терминал или командную строку.
   - Перейдите в директорию с проектом.
   - Убедитесь, что у вас установлен Python и менеджер пакетов `pip`.
   - Выполните команду для установки всех необходимых библиотек:
     ```bash
     pip install -r requirements.txt
     ```

## Использование

1. **Запуск приложения:**
   - Запустите скрипт `main.py`:
     ```bash
     python main.py
     ```
   - Откроется графический интерфейс приложения.

2. **Настройка и отправка писем:**
   - Введите тему письма в соответствующее поле.
   - Введите текст письма в текстовое поле.
   - Выберите файл Excel с адресами электронной почты, нажав кнопку "Выбрать" рядом с полем "Файл Excel".
   - При необходимости, добавьте вложения, нажав кнопку "Добавить файлы" и выбрав нужные файлы.
   - Если необходимо удалить файл из вложений, выберите его в списке и нажмите кнопку "Удалить выбранное".
   - Нажмите кнопку "Отправить письма", чтобы отправить письма всем адресатам, указанным в файле Excel.

## Формат файла Excel

Файл Excel должен содержать столбец с именем `Email`, в котором будут перечислены адреса электронной почты получателей. Пример:

| Email             |
| ----------------- |
| example1@mail.com |
| example2@mail.com |
| example3@mail.com |

## Замечания

- Приложение использует SMTP-сервер Gmail. Убедитесь, что у вас есть учетная запись Gmail с включенным доступом для приложений с более низким уровнем безопасности, либо используйте [пароль приложения](https://support.google.com/accounts/answer/185833?hl=ru) для отправки писем.
- Не забудьте обновить переменные `sender_email` и `sender_password` в коде, если вы планируете использовать другой адрес электронной почты.
